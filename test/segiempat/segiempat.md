# Segiempat
## Soal
Diberikan sebuah integer n. Buatlah segiempat dengan ukuran n x n dengan menggunakan karakter * dan #. Lihat contoh untuk mengetahui cara membuat segiempat.
## Input
- n: integer (1 ≤ n ≤ 100)
## Output
- Sebuah string yang membentuk segiempat.
## Contoh Input 1
```
1
```
## Contoh Output 1
```
*
```
## Contoh Input 2
```
2
```
## Contoh Output 2
```
**
**
```
## Contoh Input 3
```
3
```
## Contoh Output 3
```
***
*#*
***
```
## Contoh Input 4
```
4
```
## Contoh Output 4
```
****
*##*
*##*
****
```
