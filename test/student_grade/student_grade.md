# Student Grade
## Soal
Diberikan sebuah integer a yang menyatakan nilai mahasiswa. Tentukan grade yang diterima mahasiswa tersebut. Diketahui ada 7 buah grade:
- A: Nilai ≥ 80 dan ≤ 100
- AB: Nilai ≥ 73 dan ≤ 79
- B: Nilai ≥ 65 dan ≤ 72
- BC: Nilai ≥ 57 dan ≤ 64
- C: Nilai ≥ 50 dan ≤ 56
- D: Nilai ≥ 35 dan ≤ 49
- E: Nilai ≥ 0 dan ≤ 34
Keluarkan "INVALID" bila integer a tidak valid.
## Input
- t: integer (-10 ≤ t ≤ 110)
## Output
- Sebuah string yang menunjukkan grade mahasiswa.
## Contoh Input 1
```
76
```
## Contoh Output 1
```
AB
```
## Contoh Input 2
```
105
```
## Contoh Output 2
```
INVALID
```
